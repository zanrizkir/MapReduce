Cara Kerja MapReduce

Memproses data besar dengan efisien dalam tiga fase utama

Map Phase

Memproses input data dan menghasilkan pasangan key-value. Setiap mapper bekerja secara independen pada bagian data yang berbeda.

  • Parallel Processing
  • Key-Value Generation
  • Data Transformation

Shuffle Phase

Mengelompokkan dan mengurutkan data berdasarkan kunci. Mengumpulkan semua nilai yang terkait dengan kunci yang sama.

  • Data Grouping
  • Key Sorting
  • Data Organization

Reduce Phase

Mengolah data yang telah dikelompokkan untuk menghasilkan output akhir. Melakukan agregasi dan perhitungan final.

  • Data Aggregation
  • Final Computation
  • Result Generation
